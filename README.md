# Personal Expense Tracker

## Description
A web-based Personal Expense Tracker that helps users manage their income, expenses, budgets, and categories. It provides a dashboard to monitor financial activities and spending.

## Features
- User Login & Registration
- Dashboard
- Add/Edit/Delete Expenses
- Add/Edit/Delete Income
- Budget Management
- Expense Categories

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## Prerequisites
- Node.js
- MySQL (XAMPP/phpMyAdmin)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/sandhiyap25/expense-tracker.git
```

2. Open the project folder:

```bash
cd expense-tracker
```

3. Install dependencies:

```bash
npm install
```

4. Create a MySQL database named:

```
expense_tracker
```

5. Import the `expense_tracker.sql` file into the database.

6. Start the server:

```bash
node server.js
```

7. Open your browser and visit:

```
http://localhost:3000
```
